# Chatbot_ThuVien
Các bước cài đặt hệ thống: 
    1. Cài đặt đầy đủ Python và Ollama
    2. Tiến hành cài đặt mô hình ngôn ngữ lớn:
        - Mở "PowerShell" gõ lệnh: "ollama pull qwen2.5:1.5b"
        - Sau đó, nhập lệnh "ollama serve để mở port"
    3. Triển khai hệ thống:
        - Mở folder "SanPham" giữ shift và chuột phải để hiện lên mục "Mở ở đây với cửa sổ PowerShell" 
        - Nhập lệnh "py run.py" sau đó nếu trình duyệt tự động mở cửa sổ mới thì chứng minh hệ thống đã khởi động thành công.
    Dự án còn nhiều thiếu sót và sai lệch nhiều do mô hình ngôn ngữ lớn chưa được tối ưu. Vì vậy, chúng em xin nhận nhiều góp ý và yêu cầu chỉnh sửa từ phía thầy, cô và người dùng để hoàn thiện sản phẩm hơn trong tương lai.

    Chúng em xin chân thành cám ơn!
